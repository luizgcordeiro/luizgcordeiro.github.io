1.3. Topological groupoids

Definition 1.3.1 A topological groupoid is a groupoid $\mathcal{G}$ endowed with a topology which makes the product map $\mathcal{G}^{(2)}\to G$ and the inversion $\mathcal{G}\to \mathcal{G}$ continuous (where we consider the product topology on $\mathcal{G}^{(2)}$).

Example 1.3.2 Any topological group, seen as a groupoid, is a topological groupoid.

Example 1.3.3 If $R$ is an equivalence relation on a topological space $X$, then $R$ is a topological groupoid when endowed with the topology induced by the product topology of $X\times X$, however in general this is not the most suitable topology (see Example ).

Example 1.3.4 Let $G$ be a topological group acting continuously on a topological space $X$ (that is, the action map $G\times X\to X$, $(g,x)\mapsto gx$ is continuous). Then the transformation groupoid $G\ltimes X$ endowed with the product topology of $G\times X$ is a topological groupoid.

Example 1.3.5 Let $\mathcal{G}$ be any groupoid and assume $\mathcal{G}^{(0)}$ is endowed with some (arbitrary) topology. Define a topology on $\mathcal{G}$ where $A\subseteq \mathcal{G}$ is open if and only if $A\cap\mathcal{G}^{(0)}$ is open in $\mathcal{G}^{(0)}$. This makes $\mathcal{G}$ a topological groupoid. Indeed, $A^{-1}\cap\mathcal{G}^{(0)}=A\cap\mathcal{G}^{(0)}$ for all $A\subseteq\mathcal{G}$, which implies that the inversion is continuous, so let us check the product is continuous: Let $(a,b)\in \mathcal{G}^{(2)}$ and $U$ any open set containing $ab$. There are a few possibilities:

  • If $a,b\not\in\mathcal{G}^{(0)}$, then $\left\{a\right\}$ and $\left\{b\right\}$ are open and $\left\{a\right\}\left\{b\right\}=\left\{ab\right\}\subseteq U$;
  • If $a\not\in \mathcal{G}^{(0)}$ but $b\in\mathcal{G}^{(0)}$, then $\left\{a\right\}$ and $\mathcal{G}^{(0)}$ are open, respectively, and $\left\{a\right\}\mathcal{G}^{(0)}=\left\{ab\right\}\subseteq U$;
  • The case $a\in \mathcal{G}^{(0)}$ and $b\not\in\mathcal{G}^{(0)}$ is similar to the previous one;
  • If both $a,b\in \mathcal{G}^{(0)}$, then $a=b\in U\cap\mathcal{G}^{(0)}$ and $(U\cap\mathcal{G}^{(0)})(U\cap\mathcal{G}^{(0)})=U\cap\mathcal{G}^{(0)}\subseteq U$;
In any case there are open sets $A$ and $B$ containing $a$ and $b$, respectively, such that $AB\subseteq U$, which proves that the product is continuous.

Definition 1.3.6 A groupoid is étale if the source map $\mathfrak{s}:\mathcal{G}\to\mathcal{G}^{(0)}$ map is a local homeomorphism.

Étale groupoids were first considered in depth by Renault in [], in connection with $C^*$-algebras, as those groupoids which have open unit space and admit a left Haar system. We initially adopt the current (equivalent) working definition; see Definition 2.6 and Proposition 2.8 of [], and Theorem below.

Example 1.3.7 Every topological space, seen as a unit groupoid (Example ), is étale.

Example 1.3.8 If $G$ is a topological group acting continuously on a topological space $X$, then the transformation groupoid $G\ltimes X$ (endowed with the product topology) is étale if and only if $G$ is discrete.

Example 1.3.9 Let $R$ be an equivalence relation on a topological space $(X,\nu)$.

In general, the product topology makes $R$ a non-étale topological groupoid, the simplest example being $R=X\times X$ when $X$ has any non-isolated point (see Proposition ).

On the other hand, suppose that $(X,\nu)$ is Hausdorff and $\tau$ is any topology on $R$ for which:

  1. $(R,\nu)$ is a compact étale topological groupoid;
  2. The range map $\ra:(R,\tau)\to(X,\nu)$ is continuous.
Then $\tau$ is induced from the product topology of $(X,\nu)\times(X,\nu)$ (see [, Proposition 3.2]). In particular, in this case the topology $\nu$ of $X$, identified with $R^{(0)}$, is induced from $\tau$. (Compare this with Example in the Borel setting.)

The initial problem we will deal with is showing that $\mathcal{G}^{(0)}$ is open in $\mathcal{G}$ whenever $\mathcal{G}$ is étale. This is not immediate from the étale condition, which deals, in principle, only with the relative topology of $\mathcal{G}^{(0)}$ induced by that of $\mathcal{G}$.

Note that if $\mathcal{G}$ is a topological groupoid and $A$ is open in $\mathcal{G}$, then $A^{-1}$ is immediately open (since the inversion $a\mapsto a^{-1}$ is a homeomorphism), however it is not necessarily true that $AB$ is open when $A$ and $B$ are open (e.g. in Example ).

Proposition 1.3.10 If $\mathcal{G}$ is an étale groupoid, then the open bisections of $\mathcal{G}$ form a basis for its topology. Moreover, if $A$ is any open bisection of an étale groupoid $\mathcal{G}$ then the source map restricts to a homeomorphism $\mathfrak{s}|_A:A\to \mathfrak{s}(A)$ (and similarly for the range map). In particular, $\mathfrak{s}^{-1}(x)$ is discrete for all $x\in\mathcal{G}^{(0)}$.

Proof

If $I:\mathcal{G}\to\mathcal{G}$, $I(a)=a^{-1}$ is the inversion map, then $\mathfrak{r}=\mathfrak{s}\circ I$, so both $\mathfrak{s}$ and $\mathfrak{r}$ are local homeomorphisms and it follows that open bisections form a basis for the topology of $\mathcal{G}$. Since the source map restricts to a bijective local homeomorphism on any open bisection it actually restricts to a homeomorphism, as desired.

The last statement follows from the fact that if $A$ is a bisection and $x\in\mathcal{G}^{(0)}$ then $\mathfrak{s}^{-1}(x)\cap A$ is either empty or a singleton.

Theorem 1.3.11 [, Theorem 5.18] Let $\mathcal{G}$ be a topological groupoid and let $m:\mathcal{G}^{(2)}\to G$, $m(a,b)=ab$ be the product map. The following are equivalent:

  1. $\mathcal{G}$ is étale.
  2. $\mathcal{G}^{(0)}$ is open and $m$ is a local homeomorphism.
  3. $\mathcal{G}^{(0)}$ is open in $\mathcal{G}$ and $m$ is an open map (or equivalently, the product of open sets in $\mathcal{G}$ is open).
  4. $\mathcal{G}^{(0)}$ is open and the source map $\mathfrak{s}:\mathcal{G}\to\mathcal{G}^{(0)}$ is an open map.
Proof

(1)$\Rightarrow$(2): Assume $\mathcal{G}$ is étale, and let $\pi:\mathcal{G}^{(2)}\to\mathcal{G}$, $\pi(a,b)=b$, which is immediately continuous. Let us show that $\pi$ is a local homeomorphism: If $A$ and $B$ are open bisections in $\mathcal{G}$, then $\pi((A\times B)\cap\mathcal{G}^{(2)})=B\cap\mathfrak{r}|_B^{-1}(\mathfrak{s}(A)\cap \mathfrak{r}(B))$. Since $\mathfrak{r}(B)$ and $\mathfrak{s}(A)$ are open in $\mathcal{G}^{(0)}$ and $\mathfrak{r}|_B:B\to\mathfrak{r}(B)$ is a homeomorphism we conclude that $\pi$ is open, and since it is injective on $(A\times B)\cap\mathcal{G}^{(2)}$, then $\pi$ is a local homeomorphism, as we wanted. We now have $\mathfrak{s}\circ m=\mathfrak{s}\circ\pi$, and both $\mathfrak{s}$ and $\pi$ are local homeomorphisms, so $m$ is a local homeomorphism as well. In particular, if $A$ is a bisection then $\mathfrak{s}(A)=m((A\times A^{-1})\cap \mathcal{G}^{(2)})$ is open in $\mathcal{G}$, so since open bisections cover $\mathcal{G}$ and $\mathcal{G}^{(0)}=\mathfrak{s}(\mathcal{G})$, $\mathcal{G}^{(0)}$ is open in $\mathcal{G}$.

The implication (2)$\Rightarrow$(3) is trivial.

(3)$\Rightarrow$(4): Assuming (3), if $A$ is open then $\mathfrak{s}(A)=(A^{-1}A)\cap \mathcal{G}^{(0)}$ is open as well.

(4)$\Rightarrow$(1): Let $a\in\mathcal{G}$. The map

\[ f:\left\{(g,h)\in\mathcal{G}:\so(g)=\so(h)\right\}\to\mathcal{G},\qquad f(g,h)=gh^{-1} \]
is continuous, since $\mathcal{G}$ is a topological groupoid. We have $f(a,a)=\ra(a)\in\mathcal{G}^{(0)}$, and $\mathcal{G}^{(0)}$ is open. Thus there exist open sets $C,D\subseteq\mathcal{G}$ containing $a$ such that
\[ CD^{-1}=f\left(\left\{(c,d)\in C\times D:\so(c)=\so(d)\right\}\right)\subseteq\mathcal{G}^{(0)}. \]
This implies that the source map is injective on the neighbourhood $C\cap D$ of $a$. Therefore, the source map is continuous (again, since $\mathcal{G}$ is a topological groupoid), open, and locally injective, hence a local homeomorphism.

The next examples prove that a topological groupoid may have open unit space but the product of two open sets might not be open and vice-versa.

Example 1.3.12 Given a topological group $G$, seen as a topological groupoid, the product of open sets of $G$ is always open, however the unit space is the singleton $\left\{1\right\}$, so $G$ is étale if and only if it is discrete.

Example 1.3.13 Given an infinite set $P$ of integers, we consider the topology on $\mathbb{Z}$ generated by the sets $p\mathbb{Z}+a$, where $a\in\mathbb{Z}$ and $p\in P$. This makes $\mathbb{Z}$ a non-discrete (i.e., non-étale), countable, second-countable, zero-dimensional, non-locally compact, Hausdorff topological group.

Example 1.3.14 Let $\mathcal{G}$ be a groupoid such that $\mathcal{G}^{(0)}$ is endowed with some topology, and consider $\mathcal{G}$ with the topology induced by $\mathcal{G}^{(0)}$ as in Example . The unit space $\mathcal{G}^{(0)}$ is open, so $\mathcal{G}$ is étale if and only if the product of open sets is open. This happens if and only if for every $a\in\mathcal{G}\setminus\mathcal{G}^{(0)}$, $\mathfrak{s}(a)$ is isolated in $\mathcal{G}^{(0)}$.

Example 1.3.15 As a particular case of the previous example, let $\mathbb{Z}_2=\left\{0,1\right\}$ be the group with two elements, and consider the interval $[0,1]$ as a unit groupoid. Endow the (algebraic) product groupoid $\mathbb{Z}_2\times [0,1]$ with the topology whose open sets are those $A\subseteq\mathbb{Z}_2\times[0,1]$ such that $\left\{x\in[0,1]:(0,x)\in A\right\}$ is open in $[0,1]$. Then $\mathbb{Z}_2\times[0,1]$ is a non-étale, Hausdorff topological groupoid with open unit space. Indeed, $\left\{(1,0)\right\}$ is open in $\mathbb{Z}_2\times[0,1]$, but $\left\{(1,0)\right\}\left\{(1,0)\right\}=\left\{(0,0)\right\}$ is not.

Finally, whenever dealing with topological structures we will be mostly interested in maps which preserve the topology up to the correct degree, so we define precisely what we mean by topological groupoid morphisms below.

Definition 1.3.16 A topological groupoid morphism is a continuous groupoid morphism of topological groupoids. A topological groupoid isomorphism is a groupoid morphism which is also a homeomorphism.
