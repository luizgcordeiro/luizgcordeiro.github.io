1.2. Inverse semigroups

\subsection*{Nomenclature and notation} Before we start this section, let us fix some nomenclature that will be used throughout the text: Let $(L,\leq)$ be a poset: \begin{enumerate} \item The \emph{join}\index{Join (in a poset)} of a subset $F\subseteq L$ is its supremum, and we denote it by $\bigvee F$ if it exists. If $F=\left\{a_1,\ldots,a_n\right\}$ is finite, we also denote $\bigvee F=a_1\lor\cdots\lor a_n$.\nomenclature[O]{$\bigvee F$, $a\lor b$}{Joins in a poset} \item The \emph{meet}\index{Meet (in a poset)} of a subset $F\subseteq L$ is its infimum, and we denote it by $\bigwedge F$ if it exists. If $F=\left\{a_1,\ldots,a_n\right\}$ is finite, we also denote $\bigwedge F=a_1\land\cdots\land a_n$.\nomenclature[O]{$\bigwedge F$, $a\land b$}{Meets in a poset} \end{enumerate} We say that $(L,\leq)$ is a \begin{enumerate} \item[3.] \emph{$\wedge$-semilattice} (read ``meet semilattice'') if it admits arbitrary finite meets.\index{Semilattice!$\land$-semilattice (as a poset)} \item[4.] \emph{lattice}, if it admits arbitrary finite meets and joins.\index{Lattice} %\item \emph{weak $\wedge$-semilattice} if for all $s,t\in L$, there is a finite (possibly empty) subset $F\subseteq L$.\index{Semilattice!Weak $\land$-semilattice} \end{enumerate} We say that the poset $L$ has a \emph{zero} if it has a minimum, denoted by $0=\bigwedge L$.\nomenclature[O]{$0$ (in a poset)}{The minimum of a poset}\index{Zero (of a poset)} \subsection{Inverse semigroups}

Definition 1.2.1 A semigroup is a set $S$ endowed with an associative binary operation $(s,t)\mapsto st$. A semigroup is

  1. regular if for every $s\in S$ there exists an element $t\in S$, called an inverse of $s$, satisfying $sts=s$ and $tst=t$.
  2. an inverse semigroup if it is regular and every $s\in S$ admits a unique inverse. In this case we denote it as $s^*$.
  3. a monoid if it has an identity or unit, that is, an element $1\in S$ (necessarily unique) satisfying $1s=s1=s$ for all $s\in S$.
A subsemigroup of a semigroup $S$ is a subset $T\subseteq S$ which is closed under the semigroup operation. A sub-inverse semigroup of an inverse semigroup is a subsemigroup which is closed under inverses.

\begin{definition}\index{Morphism (of semigroups)}\index{Isomorphism (of semigroups)}\label{definitionmorphismofsemigroups} A \emph{morphism} of semigroups is a map $\theta:S\to T$ satisfying $\theta(s_1s_2)=\theta(s_1)\theta(s_2)$ for all $s_1,s_2\in S$. Moreover, $\theta$ is an \emph{isomorphism} if it is invertible and $\theta^{-1}$ is also a morphism. \end{definition} As usual, we extend the product notation of elements of a semigroup $S$ to subsets of $S$, namely \[FG=\left\{fg:(f,g)\in F\times G\right\},\quad tF=\left\{tf:f\in F\right\}\quad\text{and}\quad Ft=\left\{ft:f\in F\right\}\nomenclature[S]{$FG$}{}\nomenclature[S]{$tF$, $Ft$}{}\] whenever $F,G\subseteq S$ and $t\in S$. If $S$ is an inverse semigroup, we also denote \[F^*=\left\{f^*:f\in F\right\}.\nomenclature[S]{$F^*$}{}\] \begin{remark}\label{remarksubinversesemigroups} If $S$ is an inverse semigroup and $T$ is a subsemigroup which is regular, then every element $t\in T$ admits an inverse $t'\in T$. However since inverses are unique in $S$ then $t'=t^*$. Therefore regular sub-semigroups of inverse semigroups are sub-inverse semigroups. \end{remark} \begin{example}\index{Semilattice!$\land$-semilattice (as a semigroup)} Every $\land$-semilattice is an inverse semigroup with the meet as the operation: $xy=x\land y$. In this case, every element is its own inverse: $x^*=x$. We will always regard $\land$-semilattices as inverse semigroups with this operation. \end{example} \begin{example} If $S$ is an inverse semigroup and $s,t\in S$, then the equation $sts=s$ might be satisfied even though $t$ is not the inverse of $s$. As a particular case of the previous example, let $X$ be a nonempty set and consider the power set $\mathcal{P}(X)$ of $X$, ordered by set inclusion. Then $\mathcal{P}(X)$ is a $\land$-semilattice, and hence an inverse semigroup, under set intersection: $AB=A\land B=A\cap B$ for all $A,B\in\mathcal{P}(X)$. However, $\varnothing X\varnothing=\varnothing$, even though $X$ is not the inverse of $\varnothing$. \end{example} \begin{example} Not every regular semigroup is an inverse semigroup. For example, if $S$ is any set we endow it with the operation $st=s$ for all $s,t\in S$, which makes it regular but not an inverse semigroup if $|S|\geq 2$. In fact, any two elements are inverses of each other in this semigroup. \end{example} \begin{example}\nomenclature[S]{$\mathcal{I}(X)$}{Semigroup of partial bijections of $X$}\nomenclature[Z]{$\operatorname{dom}(f)$,$\operatorname{ran}(f)$}{Domain and range of a partial bijection $f$.} Let $X$ be a set. Let $\mathcal{I}(X)$ be the set of all partial bijections of $X$, i.e., the set of all bijections $f:\operatorname{dom}(f)\to\operatorname{ran}(f)$, where $\dom(f)$ and $\ran(f)$ are subsets of $X$. We make it a semigroup with the canonical partial composition of maps, namely, given $f,g\in\mathcal{I}(X)$, \begin{enumerate} \item $\dom(gf)=f^{-1}(\dom(g)\cap\ran(f))$; \item $\ran(gf)=g(\dom(g)\cap\ran(f))$; \item $(gf)(x)=g(f(x))$ for all $x\in\dom(gf)$. \end{enumerate} This makes $\mathcal{I}(X)$ an inverse semigroup. Namely, the inverse of $f\in\mathcal{I}(X)$ is the usual inverse function $f^*=f^{-1}:\ran(f)\to\dom(f)$. $\mathcal{I}(X)$ serves as an analogue to permutation groups, and in fact a version of Cayley's Theorem, called the \emph{Vagner-Preston Theorem} (see Theorem~\ref{theoremwagnerpreston}) holds: every inverse semigroup is isomorphic to a sub-inverse semigroup of some $\mathcal{I}(X)$. the \emph{graph} of $f$ is the set \[\operatorname{graph}(f)=\left\{(f(x),x):x\in X\right\}.\] \end{definition} \begin{remark} Usually (see \cite[p.\ 11]{MR1940513} or \cite[Section 8]{MR0453532}), Recall that a \emph{relation} (between sets $X$ and $Y$) is a subset $A\subseteq Y\times X$. The \emph{composition} of relations $A\subseteq Y\times X$ and $B\subseteq Z\times Y$ is the relation \[B\circ A=\left\{(z,x):\text{there exists }y\in Y\text{ such that }(z,y)\in B\text{ and }(y,x)\in A\right\}.\] (Note that if $R$ is an equivalence relation, regarded as a groupoid, and $A,B\subseteq R\subseteq X\times X$, then the product $BA$ coincides with the composition $B\circ A$.) If $f:X\to Y$ and $g:Y\to Z$ are functions, then $\operatorname{graph}(f)\subseteq Y\times X$ and $\operatorname{graph}(g)\subseteq Z\times Y$ are relations, and \[\operatorname{graph}(g\circ f)=\operatorname{graph}(g)\circ\operatorname{graph}(f)\] i.e., graph preserves composition. In particular, if $S$ is an inverse semigroup then $E(S)=\left\{s^*s:s\in S\right\}$. If $S$ is a monoid then $1=1^2\in E(S)$. \begin{example}\label{exampleidempotentssets}\nomenclature[Z]{$\id_A$}{Identity function of a set $A$} Given a set $A$, let $\id_A:A\to A$ be the identity function of $A$. Then if $X$ is a set, the idempotent set of $\mathcal{I}(X)$ is \[E(\mathcal{I}(X))=\left\{\id_A:A\subseteq X\right\}.\] Also note that $\id_{A\cap B}=\id_A\id_B$ whenever $A,B\subseteq X$, so by identifying subsets of $X$ with their respective identity functions, the semigroup operation of $\mathcal{I}(X)$ corresponds to intersection of sets. Moreover, $f^*f=\id_{\dom(f)}$ and $ff^*=\id_{\ran(f)}$, so $f^*f$ corresponds to the domain of $f$ and $ff^*$ corresponds to the range of $f$. \end{example} We now prove that $E(S)$ is a commutative sub-inverse semigroup of an inverse semigroup $S$, and this is an important fact for calculations which will be performed throughout the next sections and chapters. \begin{theorem}[{\cite[Theorem 5.1.2]{MR0466355}, \cite[Theorem 1.1.3]{MR1694900}}]\label{theoreminverseiffescommute} A regular semigroup $S$ is inverse if and only if the elements of $E(S)$ commute. In this case, $E(S)$ is a sub-inverse semigroup of $S$. \end{theorem} \begin{proof} First assume that $S$ is inverse and let $e,f\in E(S)$. Let $x=(ef)^*$ be the unique inverse of $ef$. Then \[(ef)(fxe)(ef)=(ef)x(ef)=ef\quad\text{and}\quad (fxe)(ef)(fxe)=f(xefx)e=fxe\] so $(fxe)=(ef)^*=x$. Then we obtain \[x^2=(fxe)(fxe)=f(xefx)e=fxe=x\] so $x$ is idempotent, and therefore it is its own inverse, that is, $x=fxe=ef$ is idempotent. Similarly, $fe$ is idempotent, so \[fe=(fe)^2=f(ef)e=fxe=x=ef\] Conversely, assume elements of $E(S)$ commute and let $t_1$ and $t_2$ be inverses of some $s\in S$. Since $st_1$, $st_2$, $t_1s$ and $t_2s$ are idempotents, then \begin{align*} t_1&=t_1st_1=t_1(st_2)(st_1)=t_1(st_1)(st_2)=t_1st_2=(t_1s)(t_2s)t_2=(t_2s)(t_1s)t_2\\ &=t_2st_2=t_2 \end{align*} so $S$ is inverse. If elements of $E(S)$ commute, it is easy to see $E(S)$ is closed under product, and since every element of $E(S)$ is its own inverse then $E(S)$ is a sub-inverse semigroup of $S$.\qedhere \end{proof} An easy application of this result allows us to conclude that inverses in inverse semigroups respect the usual laws of products. \begin{corollary}\label{theorempropertiesofinversesinsemigroups} Let $S$ be an inverse semigroup. The next proposition gives a better characterization of this element in terms of the order: The next simple proposition gives a simple characterization of zeros in inverse semigroups as absorbing elements. \begin{proposition}\label{propositionzerosemigroup}\index{Zero (in a semigroup)} An element $p$ of an inverse semigroup $S$ is a zero (minimum) of $S$ if and only if it is absorbing (that is, $ps=p$ for all $s\in S$). \end{proposition} \begin{proof} If $p$ is absorbing, then for all $s$, $p=ps^*s$, so $p\leq s$ and $p=\bigwedge S$; In the other direction, if $p=\bigwedge S$ then $p\leq p^*p$, which is idempotent, and so $p\in E(S)$ by Theorem \ref{theoremorder}(c) and in particular $p=p^*$. For all $s\in S$, $p=sp^*p=sp$, so $p$ is absorbing.\qedhere \end{proof} Therefore, we denote any absorbing element of an inverse semigroup $S$ by $0$ and call it a \emph{zero}, which agrees with the poset definition at the beginning of this section.\nomenclature[S]{$0$ (in an inverse semigroup)}{The absorbing element of an inverse semigroup}\index{Zero (of an inverse semigroup)} \begin{example} In general, a zero for $E(S)$ is not necessarily a zero for $S$ when $S$ is an inverse semigroup. For example, if $G$ is a nontrivial group then $E(G)=\left\{1\right\}$, and $1$ is the minimum of $E(G)$ but not of $G$. \end{example} We will now prove a generalization of Cayley's Theorem in group theory, which allows us to think of $\mathcal{I}(X)$ as a sort of ``canonical'' inverse semigroup. The next proposition summarizes some basic properties of morphisms of semigroups which will be used throughout the text. \begin{proposition}\label{propositionpropertiesofmorphismsofsemigroups} Let $\theta:S\to T$ be a morphism of semigroups. \begin{enumerate} \item[(a)] $\theta(S)$ is a subsemigroup of $T$; \item[(b)] If $\theta$ is invertible then it is immediately an isomorphism; \item[(c)] If $S$ is regular then $\theta(S)$ is regular; \item[(d)] If $S$ is regular and $T$ is an inverse semigroup then $\theta(S)$ is a sub-inverse semigroup of $T$. \item[(e)] If $S$ is inverse and $\theta$ is surjective, then $\theta(E(S))=E(T)$, and $T$ is an inverse semigroup; \item[(f)] If $S$ and $T$ are inverse semigroups then $\theta(s^*)=\theta(s)^*$ for all $s\in S$. \end{enumerate} \end{proposition} \begin{proof} (a) and (b) can be proven in the same manner as the analogous statements for groups. (c) is clear and (d) follows from Remark \ref{remarksubinversesemigroups} and the previous items, and (f) follows easily from the uniqueness of inverses in inverse semigroups. The only item we need to prove then is (e), so suppose $S$ is inverse, $\theta$ is surjective and let $u=\theta(s)\in E(T)$, where $s\in S$. Letting $e=(ss^*)(s^*s)\in E(S)$, we obtain \begin{align*} \phi(e)=u\phi(s^*s^*)u=u^2\phi(s^*s^*)u^2=\phi(s^2(s^*)^2s^2)=\phi(s^2)=u^2=u \end{align*} so $u=\phi(e)\in\phi(E(S))$. In particular, $E(T)$ is commutative and $T=\theta(S)$ is regular, so $T$ is an inverse semigroup by Theorem \ref{theoreminverseiffescommute}.\qedhere \end{proof} \begin{theorem}[{Vagner-Preston Theorem, \cite{MR0048425,MR0064038}}]\label{theoremwagnerpreston} Every inverse semigroup $S$ is isomorphic to a sub-inverse semigroup of $\mathcal{I}(X)$ for some set $X$. \end{theorem} \begin{proof} For every $s\in S$, let $D_{s^*}=\left\{t\in S:tt^*\leq s^*s\right\}$, and define a map $\alpha_s:D_{s^*}\to D_s$ by $\alpha_s(z)=sz$. To check that $\alpha_s$ is well-defined, note that for all $z\in D_{s^*}$, \[\alpha_s(z)\alpha_s(z)^*=(sz)(sz)^*=szz^*s^*\leq ss^*,\] so $\alpha_s(z)=sz\in D_s$. The verification that $\alpha_{s^*}=\alpha_s^{-1}$ follows easily from the definition of the order on $S$. In order to prove that $\alpha_s\circ\alpha_t=\alpha_{st}$, we first compare their domains, that is, we show that \[\alpha_t^{-1}(D_t\cap D_{s^*})=D_{(st)^*}\] If $z\in D_{(st)^*}$, then $zz^*\leq (st)^*(st)=t^*s^*st\leq t^*t$ (because $t^*s^*s\leq t^*$), so $z\in D_{t^*}$, and \[(\alpha_t(z))(\alpha_t(z))^*=(tz)(tz)^*=tzz^*t^*\leq t(st)^*(st)t^*=tt^*s^*stt^*\leq s^*s\] that is, $z\in\alpha_t^{-1}(D_t\cap D_{s^*})$. Conversely, if $z\in \alpha_t^{-1}(D_t\cap D_{s^*})$, then $\alpha_t(z)\in D_{s^*}$ which implies $\alpha_t(z)=s^*s\alpha_t(z)$, so \[zz^*(st)^*(st)=zz^*t^*s^*st=z\alpha_t(z)^*s^*st=z\alpha_t(z)^*t=zz^*t^*t=zz^*\] and therefore $z\in D_{(st)^*}$. It is then clear that $\alpha_{st}=\alpha_s\circ\alpha_t$. Finally, for the injectivity of $\alpha$, simply note that if $\alpha_s=\alpha_t$ then $s=\alpha_s(s^*s)=\alpha_t(s^*s)=ts^*s$ so $s\leq t$ and symmetrically, $t\leq s$, so $\alpha:S\to\mathcal{I}(S)$ is an isomorphism from $S$ to a sub-inverse semigroup of $\mathcal{I}(S)$.\qedhere \end{proof} The map $\alpha$ will be used in later chapters, so we give it a name. \begin{definition}\index{Canonical left action of $S$ on itself}\label{definitioncanonicalleftaction} The map $\alpha:S\to\mathcal{I}(S)$ is called the \emph{canonical left action of $S$ on itself}. \end{definition} \begin{remark} For inverse semigroups with zero, it is natural to ask that morphisms preserve zeros, however this is not the case for the canonical left action. Namely, if $S$ has a zero and $\alpha:S\to\mathcal{I}(S)$ is the canonical left action, then $\theta(0)=\operatorname{id}_{\left\{0\right\}}$, which is not the empty function. However, this is easily remedied by noticing that $0$ is fixed by $\alpha_s$ for all $s\in S$, and so we can restrict each $\alpha_s$ to the complement of $\left\{0\right\}$ and obtain again an injective morphism $\alpha_0:S\to\mathcal{I}(S\setminus\left\{0\right\})$, $\alpha_0(s)=\alpha_s|_{S\setminus\left\{0\right\}}$, which in this case will satisfy $\alpha_0(0)=\varnothing$. \end{remark} We can use the Vagner-Preston Theorem in order to prove that all inverse semigroups are isomorphic to some semigroup of partial isometries of some C*-algebra (this was first proved in \cite{MR785726}). Indeed, it is enough to prove this for $\mathcal{I}(X)$, where $X$ is a set. Consider the Hilbert space $H=\ell^2(X)$ with the standard basis $\left\{\delta_x:x\in X\right\}$. Given $f\in\mathcal{I}(X)$, we define $T_f\in B(H)$ by \[T_f(\delta_x)=\begin{cases}\delta_{f(x)},&\text{if }x\in\dom(f),\\ 0,&\text{otherwise,}\end{cases}\] and the map $f\mapsto T_f$ is an injective semigroup morphism. \subsection{Distributive semigroups} Let $X$ be a set and $f,g$ two partial bijections of $X$. Oftentimes we are interested in considering a common upper bound of $f$ and $g$, that is, a new function which extends both $f$ and $g$. This is possible, for example, if $f$ and $g$ coincide on the intersection of their domains, and $f^*$ and $g^*$ also coincide on the intersection of their domains. We will deal with this problem in the more abstract setting of general inverse semigroups (as opposed to $\mathcal{I}(X)$). \begin{proposition} Suppose $S$ is an inverse semigroup and $s,t\in S$ admit a common upper bound (that is, there is some $z\in S$ with $s\leq z$ and $t\leq z$). Then $s^*t\in E(S)$ and $st^*\in E(S)$. \end{proposition} \begin{proof} Let $z\in S$ be a common upper bound of $s$ and $t$. Then $s^*t\leq z^*z$, which is an idempotent and thus $s^*t\in E(S)$. The same argument with $s^*$ and $t^*$ in place of $s$ and $t$ yields $st^*\in E(S)$ as well.\qedhere \end{proof} \begin{definition}\label{definitioncompatibility} Two elements $s,t\in S$ are \emph{compatible} if $s^*t$ and $st^*$ belong to $E(S)$. A subset $F\subseteq S$ is called \emph{compatible} if any two elements of $F$ are compatible. \end{definition} \begin{proposition}\label{propositioncompatibility} Let $S$ be an inverse semigroup, $s,t\in S$ two compatible elements and $z\in S$. Then \begin{enumerate} \item[(a)] $s^*$ and $t^*$ are compatible; \item[(b)] $zs$ and $zt$ are compatible. \item[(c)] $s\land t$ exists, and $s\land t=st^*t=tt^*s=ts^*s=ss^*t$. \end{enumerate} \end{proposition} \begin{proof} \begin{enumerate} \item[(a)] should be clear enough. \item[(b)] We have \[(zs)^*(zt)=(s^*z^*z)t\leq s^*t\qquad\text{and}\qquad(zs)(zt)^*=z(st^*)z^*\] so $(zs)^*(zt)\in E(S)$ by Theorem \ref{theoremorder}(c), and $(zs)(zt)^*\in E(S)$ by Theorem \ref{theorempropertiesofinversesinsemigroups}(c), so $zs$ and $zt$ are compatible. \item[(c)] We just prove that $s\land t=st^*t$, and the other equalities are similar. Since, $st^*\in E(S)$, then $st^*t\leq t$, and also $st^*t\leq s$, so $st^*t$ is a lower bound of $s$ and $t$. If $p\leq s,t$, then \[p=pp^*p\leq st^*t\] so $st^*t$ is the largest lower bound of $s$ and $t$, that is, their meet.\qedhere \end{enumerate} \end{proof} \begin{example} If $X$ is a set and $f,g\in \mathcal{I}(X)$, then $f^*g\in E(\mathcal{I}(X))$ if and only if $f(x)=g(x)$ whenever $x\in\operatorname{dom}(f)\cap\operatorname{dom}(g)$, and similarly $fg^*\in E(\mathcal{I}(X))$ if and only if $f^{-1}(x)=g^{-1}(x)$ whenever $x\in\operatorname{ran}(f)\cap\operatorname{ran}(g)$. Thus, if $f$ and $g$ are compatible we may construct the partial bijection $f\lor g$, defined by \begin{itemize} \item $\operatorname{dom}(f\lor g)=\operatorname{dom}(f)\cup\operatorname{dom}(g)$; $\operatorname{ran}(f\lor g)=\operatorname{ran}(f)\cup\operatorname{ran}(g)$; \item $(f\lor g)(x)=f(x)$ if $x\in\operatorname{dom}(f)$; $(f\lor g)(x)=g(x)$ if $x\in\operatorname{ran}(g)$. \end{itemize} \end{example} We will be interested in semigroups which satisfy a distributive property for the product with respect to joins. Actually, distributivity of the product over meets is always satisfied, but there are inverse semigroups such that the product does not distribute over joins. \begin{proposition} Suppose $S$ is an inverse semigroup, $t\in S$, $F$ is a nonempty subset of $S$ and that $\bigwedge F$ exists. Then $\bigwedge(tF)$ and $\bigwedge(Ft)$ exist, and $\bigwedge(tF)=t\left(\bigwedge F\right)$ and $\bigwedge(Ft)=\left(\bigwedge F\right)t$. \end{proposition} \begin{proof} For all $f\in F$, $t(\bigwedge F)\leq tf$, so $t\left(\bigwedge F\right)$ is a lower bound of $tF$. To prove it is the largest one, assume $h$ is another lower bound of $tF$. Then $t^*h\leq t^*tf\leq f$ for all $f\in F$, which implies $t^*h\leq\bigwedge F$. Letting $p\in F$ be arbitrary, we have $h\leq tp$ and so $hh^*\leq tpp^*t^*\leq tt^*$, therefore \[h=hh^*h\leq tt^*h\leq t\left(\bigwedge F\right).\] The proof that $\bigwedge(Ft)=\left(\bigwedge F\right)t$ is similar.\qedhere \end{proof} \begin{definition}[\cite{MR3077869}]\label{definitiondistributivesemigroup}\index{Semigroup!distributive} A \emph{distributive semigroup} is an inverse semigroup $S$ such that any two compatible elements $s,t\in S$ admit a join $s\lor t$, and for every $z\in S$, $z(s\lor t)=(zs)\lor (zt)$. \end{definition} Note that in the definition we assume, in principle, only left distributivity of the product over joins, however right distributivity follows as a consequence, as we will see in Proposition \ref{propositionfinitarydistributivityofdistributivesemigroups}. \begin{example} The simplest non-distributive inverse semigroup is the \emph{diamond lattice} $M_3$: \[\begin{tikzcd} &1&\\ s\arrow[ur]&z\arrow[u]&t\arrow[ul]\\ &0\arrow[ul]\arrow[u]\arrow[ur]& \end{tikzcd}\] Here an arrow $x\to y$ denotes $x\leq y$, and the semigroup operation is the meet. $M_3$ admits binary joins, but \[z(s\lor t)=z1=z\qquad\text{however}\qquad(zs)\lor(zt)=0\lor 0=0.\] \end{example} \begin{example}\label{bisgisinfinitelydistributive} The semigroup $\operatorname{\bf Bis}(\G)$ of bisections of a groupoid $\G$ is distributive. More generally, if $\mathscr{F}$ is any (possibly infinite) compatible subset of $\operatorname{\bf Bis}(\G)$, then the join of $\mathscr{F}$ exists and coincides with the union of its elements: \[\bigvee\mathscr{F}=\bigcup\mathscr{F}=\bigcup_{A\in\mathscr{F}}A\] Moreover, in this case we have $A(\bigvee \mathscr{F})=\bigvee(A\mathscr{F})$ for any $A\in\operatorname{\bf Bis}(\G)$. We say that $\operatorname{\bf Bis}(\G)$ is \emph{infinitely distributive} (\cite{MR2277324}). (Cf.\ Definition \ref{definitionkappadistributivesemigroup}.) \end{example} \begin{example}\label{exampleixisinfinitelydistributive} If $X$ is a set then, by Example \ref{examplebisectionsofequivalencerelationarefunctions}, $\mathcal{I}(X)$ is isomorphic to $\operatorname{\bf Bis}(X\times X)$, and thus it is (infinitely) distributive by Example \ref{bisgisinfinitelydistributive} above. If $\mathscr{F}$ is any compatible subset of $\mathcal{I}(X)$, then the join of $\mathscr{F}$ is the function given by \[\bigvee \mathscr{F}:\bigcup_{f\in F}\dom(f)\to\bigcup_{f\in F}\ran(f),\qquad \left(\bigvee \mathscr{F}\right)(x)=f(x),\] where $f\in \mathscr{F}$ is chosen such that $x\in\dom(f)$ (``$\dom$'' and ``$\ran$'' stand for the domain and range of an element of $\mathcal{I}(X)$). \end{example} \begin{proposition}\label{propositionpropertiesofjoins} Let $S$ be an inverse semigroup. \begin{enumerate} \item[(a)] If $F\subseteq E(S)$ admits a join, then $\bigvee F\in E(S)$. \item[(b)] If $F\subseteq S$ admits a join, then $F^*$ also admits a join, namely $\bigvee(F^*)=\left(\bigvee F\right)^*$. \end{enumerate} \end{proposition} \begin{proof} \begin{enumerate} \item[(a)] Let $e=\bigvee F$. For every $f\in F$, $f=f^*f\leq e^*e$, so the definition of join implies $e\leq e^*e\in E(S)$, therefore $e\in E(S)$; \item[(b)] Should be clear from \ref{theoremorder}(a) and \ref{propositioncompatibility}(a).\qedhere \end{enumerate} \end{proof} In the remainder of this subsection we will consider distributive properties for sets with more than two elements. We differentiate between finite and infinite sets for the sake of clarity and to obtain appropriate notation in Chapter \ref{chaptersoficgroupoids}, and specifically Definition \ref{definitionnormalizedcountingmeasure}. \begin{definition}\nomenclature[Z]{$\# X$}{Cardinality of a finite $X$}\nomenclature[Z]{$|X|$}{Cardinality of a (arbitrary) set $X$.} Given a set $X$, we denote by $|X|$ the cardinality of $X$. In the particular case that $X$ is finite, we will use instead the symbol $\#X$ to denote its cardinality (number of elements). \end{definition} \begin{proposition}\label{propositionfinitarydistributivityofdistributivesemigroups} If $S$ is a distributive semigroup, then every finite nonempty compatible set $F\subseteq S$ admits a join, and for every $t\in S$, \[\bigvee(tF)=t\left(\bigvee F\right)\quad\text{and}\quad\bigvee (Ft)=\left(\bigvee F\right)t.\] \end{proposition} \begin{proof} This follows from induction on the cardinality $\#F$: If $\#F=1$ this is trivial, so let us just assume this holds for every compatible set with up to $n$ elements, and $\#F=n+1$. Let $z\in F$ be arbitrary. Then the distributive property implies \[z^*\left(\bigvee\left(F\setminus\left\{z\right\}\right)\right)=\bigvee\left\{z^*s:s\in F\setminus\left\{z\right\}\right\},\] a join of idempotents and hence an idempotent. Similarly, $z\left(\bigvee\left(F\setminus\left\{z\right\}\right)\right)^*\in E(S)$, so $z$ and $\bigvee\left(F\setminus\left\{z\right\}\right)$ are compatible and thus admit a join. From general order theory it should be clear that \[z\lor\left(\bigvee\left(F\setminus\left\{z\right\}\right)\right)=\bigvee F.\] Given any $t\in S$, we have \[t\left(\bigvee F\right)=t\left(z\lor\left(\bigvee\left(F\setminus\left\{z\right\}\right)\right)\right)=tz\lor\bigvee\left(tF\setminus\left\{tz\right\}\right)=\bigvee(tF).\] To obtain $\left(\bigvee F\right)t=\bigvee(Ft)$, we use the previous case with $t^*$ and $F^*$ and obtain \[t^*\left(\bigvee F^*\right)=\left(\bigvee t^* F^*\right)\] and taking inverses and applying Proposition \ref{propositionpropertiesofjoins}(b) yields $\left(\bigvee F\right)t=\bigvee(Ft)$.\qedhere \end{proof} To finish, we prove that every distributive inverse semigroup is also distributive with respect to joins and meets. We perform this in a more general setting by considering inverse semigroups which are distributive with respect to a given cardinality. For more information on cardinals, see \cite[Chapter 3]{MR1940513}. We will not make assumptions about regularity of cardinals. \begin{definition}[{\cite[p.\ 82]{MR1940513}}]\label{definitionkappadistributivesemigroup}\index{Semigroup!$\kappa$-distributive} Let $\kappa$ be a cardinal. An inverse semigroup $S$ is \emph{$\kappa$-distributive} if for every compatible set $F\subseteq S$ with $|F|<\kappa$, the supremum $\bigvee F$ exists and for all $z\in S$, \[z\left(\bigvee F\right)=\bigvee(zF).\] Note that $|zF|\leq |F|$ and $zF$ is compatible by Proposition \ref{propositioncompatibility}, and that $\left(\bigvee F\right)z=\bigvee(Fz)$ also holds by Proposition \ref{propositionpropertiesofjoins}(b) \end{definition} Proposition \ref{propositionfinitarydistributivityofdistributivesemigroups} states that distributive inverse semigroups (as in Definition \ref{definitiondistributivesemigroup}) are precisely the $\aleph_0$-distributive semigroups, where $\aleph_0=|\mathbb{N}|$ is the first infinite cardinal. The following proposition provides a sharper version (with a significantly different proof) of the main result of \cite{MR2277324}. \begin{proposition}\label{propositionkappadistributivedistributemeets} Let $S$ be a $\kappa$-distributive inverse semigroup, and suppose $F\subseteq S$ is compatible, $|F|<\kappa$. If $z\in S$ is such that $z\land f$ exists for all $f\in F$, then $z\land\left(\bigvee F\right)$ exists and \[z\land(\bigvee F)=\bigvee(z\land F),\qquad\text{ where }z\land F=\left\{z\land f:f\in F\right\}.\] \end{proposition} \begin{proof} Since $z\land F$ is bounded above by $\bigvee F$, then $z\land F$ is compatible. Clearly, $\bigvee(z\land F)$ is a lower bound of $\left\{z,\bigvee F\right\}$, so we need to prove it is the largest one: Suppose $p$ is a lower bound of $\left\{z,\bigvee F\right\}$. Then \[p=\left(\bigvee F\right)p^*p=\bigvee (Fp^*p)\] Given $f\in F$, we have $f,p\leq\bigvee F$, so $f$ and $p$ are compatible and thus $fp^*\in E(S)$, so $fp^*p\leq p\leq z$. But of course we also have $fp^*p\leq f$, so $fp^*p\leq f\land z$ for all $f\in F$. Therefore \[p=\bigvee (Fp^*p)\leq\bigvee(F\land z).\] This proves that $\bigvee(F\land z)$ is the largest lower bound of $\left\{z,\bigvee F\right\}$, that is, its meet.\qedhere \end{proof} The finitary case of this proposition becomes: \begin{corollary}\label{corollarydistributivedistributemeets} Let $S$ be a distributive inverse semigroup. If $s\lor t$, $z\land s$ and $z\land t$ exist, then $z\land(s\lor t)$ exists and \[z\land(s\lor t)=(z\land s)\lor(z\land t)\] \end{corollary} %\begin{remark} %The proof above can be easily adapted to inverse semigroups which are \emph{infinitely} distributive, and in fact in general for inverse semigroups that admit joins of compatible sets up to some given cardinality. This provides a sharper version (with a significantly different proof) of the main result of \cite{MR2277324}. %\end{remark} We will need to deal with maps that preserve more than just semigroup operations, so we define morphisms which respect the order of semigroups in a stronger manner: \nocite{MR3715848}%additive \begin{definition}\index{Morphism (of semigroups)!$\lor$-morphism}\index{Morphism (of semigroups)!$\land$-morphism} A \emph{$\lor$-morphism} of inverse semigroups is a semigroup morphism $\theta:S\to T$ such that whenever $s,t\in S$ are such that the join $s\lor t$ exists, then the join $\theta(s)\lor\theta(t)$ exists as well and $\theta(s\lor t)=\theta(s)\lor\theta(t)$. A \emph{$\land$-morphism} is defined in a similar manner. \end{definition} \begin{remark} In general, the canonical left action of a distributive inverse semigroup (which is used in the Vagner-Preston Theorem) is not a $\lor$-morphism. For example, suppose that $S$ is any distributive semigroup which admits two compatible but non-comparable elements $s$ and $t$, i.e., $s\lor t$ exists and is different from both $s$ and $t$. Let $\alpha:S\to\mathcal{I}(S)$ be the canonical left action of $S$ (Definition \ref{definitioncanonicalleftaction}). Then $(s\lor t)^*(s\lor t)$ belongs to the domain of $\alpha_{s\lor t}$, but it does not belong to the domain of $\alpha_s\lor\alpha_t$, so $\alpha_{s\lor t}$ is strictly larger than $\alpha_s\lor\alpha_t$. On the other hand, every distributive inverse semigroup $S$ admits a canonical injective $\lor$-morphism $S\to\mathcal{I}(X)$ for some set $X$. See \cite[Section 3.3]{MR3700423}. \end{remark} %Additive morphisms = lor morphisms \subsection{Relative complements} The final structural property we will have to deal with are \emph{relative complements}, which allow us to look at semigroup-theoretic analogues of set differences. \begin{definition}\index{Relative complement}\label{definitionrelativecomplementdistributive} Let $S$ be a distributive semigroup with zero and $s\leq t\in S$. A \emph{relative complement} of $s$ in $t$ is an element $c\leq t$ such that $s\land c=0$ and $t=s\lor c$ (in particular, $s$ and $c$ are compatible). \end{definition} \begin{lemma} If $S$ is a distributive inverse semigroup, $s\leq t$ in $S$ and $c,d\in S$ are relative complements of $s$ in $t$, then $c=d$. \end{lemma} \begin{proof} Since $s\lor c=t$ exists, then $s$ and $c$ are compatible and by Proposition \ref{propositioncompatibility}(c), $sc^*c=s\land c=0$, thus \[c=tc^*c=(d\lor s)c^*c=dc^*c\lor sc^*c=dc^*c\] so $c\leq d$. The reverse inequality is similar.\qedhere \end{proof} \begin{definition}\nomenclature[S]{$t\setminus s$}{Relative complement of $s$ in $t$} If it exists, we denote the relative complement of an element $s$ in $t$ ($s\leq t$), in a distributive inverse semigroup, by $t\setminus s$. \end{definition} \begin{lemma} If $S$ is an inverse semigroup, $s,t\in S$, $s\land t$ exists and $z\leq t$, then $s\land z$ exists and $s\land z=(s\land t)z^*z$ \end{lemma} \begin{proof} We already know that \[(s\land t)z^*z=(sz^*z)\land (tz^*z)=(sz^*z)\land z,\] which is a lower bound of $s$ and $z$. It is readily checked that it is the largest one.\qedhere \end{proof} In the next proposition, we initially assume that the existence of $t\setminus s$ implies $s\leq t$, however this condition is not necessary in general when we extend the notion of relative complements in \emph{Boolean} inverse semigroups (Section \ref{sectionstoneduality}). \begin{proposition}\label{propositionrelativecomplementsandoperations} Let $S$ be a distributive inverse semigroup and $s,t,z,a\in S$. \begin{enumerate} \item[(a)] If $(t\setminus s)$ exists, then $(zt)\setminus(zs)$ exists and is equal to $z(t\setminus s)$. Similarly, $(tz)\setminus(sz)$ exists and is equal to $(t\setminus s)z$. \item[(b)] If $t\land z$, $s\land z$ and $(t\setminus s)$ exist, then \[(t\land z)\setminus(s\land z)=(t\setminus s)\land z\]%=(t\land z)\setminus s\] in the sense that these elements exist and are equal. \item[(c)] If $a\leq s\lor t$, and $a\setminus s$ exists, then $a\setminus s\leq t$. \item[(d)] If $a\setminus s$ and $a\setminus t$ exist, and $s\leq t$, then $a\setminus t\leq a\setminus s$. \end{enumerate} \end{proposition} \begin{proof} \begin{enumerate} \item[(a)] Let us verify that $z(t\setminus s)$ satisfies the properties of the relative complement of $zs$ in $zt$. If $p\leq z(t\setminus s),zs$, then $z^*p\leq (t\setminus s)\land s=0$, so $p=zz^*p=0$. On the other hand, \[zt=z(s\lor(t\setminus s))=zs\lor z(t\setminus s)\] which is precisely what we need. \item[(b)] Since $t\land z$ exists and $t\setminus s\leq t$, then by the previous lemma, $(t\setminus s)\land z$ exists. Let us show it satisfies the properties of the relative complement of $(s\land z)$ in $(t\land z)$: On one hand, if $p\leq (t\setminus s)\land z$ and $p\leq (s\land z)$ then $p\leq (t\setminus s)\land s=0$, so \[\left[(t\setminus s)\land z\right]\land (s\land z)=0.\] Moreover, using the previous lemma, \[t\land z=((t\setminus s)\lor s))\land z=((t\setminus s)\land z)\lor(s\land z)\] therefore $(t\setminus s)\land z$ is the relative complement of $(s\land z)$ in $(t\land z)$.% This proves the first equality, which in turn implies the second one because %\[(t\land z)\setminus(s\land z)=(t\land z\land z)\setminus(s\land z)=\left((t\land z)\setminus s\right)\land z=(t\land z)\setminus s\] \item[(c)] Since $a,s,t\leq s\lor t$, then $a$, $s$ and $t$ (or any smaller elements) are compatible and Proposition \ref{corollarydistributivedistributemeets} allows us to compute \begin{align*} a\setminus s&=(a\setminus s)\land a=(a\setminus s)\land(s\lor t)=((a\setminus s)\land s)\lor((a\setminus s)\land t)\\ &=0\lor((a\setminus s)\land t)\leq t \end{align*} \item[(d)] We are assuming $s\leq t\leq a$, so we have \[a= s\lor(a\setminus s)\leq t\lor(a\setminus s)\] which implies $a\setminus t\leq a\setminus s$ by item (c).\qedhere \end{enumerate} \end{proof}
