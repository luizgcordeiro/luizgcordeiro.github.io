This project

Most mathematical material is written in print form, such as for books and journals, and suffers from obvious defects regarding the presentation, the most common being equations which do not fit lines, and references in far pages (and having to go back-and-forth between them).

As a colleague pointed out, it is also unreasonable that we read much of this content in an electronic device but still use format such as pdf which have those same problems.

This is a problem which has already been addressed in artistic literature, with the introduction of mobile books formats such as EPUB and MOBI, which render text adapted to whatever screen. Of course there are still some issues to be addressed (such as elements sometimes overlapping), as any new technology ought to have (recall web browser experience in the 90's).

During the Winter section of 2018 I taught a Linear Algebra course (MAT1302) at the University of Ottawa, and the textbook we used was Robert Beeze's A first course in linear algebra, which is an online textbook available at http://linear.ups.edu. This is an example of a simple but nevertheless marvelous approach to the problems above: There are no unwanted page breaks in the text; and content may be displayed or ommited dynamically, to the interest of the reader. For example, a student in social science taking a course which follows this book will probably not need to read the proofs of theorems, so they are immediatelly ommited and do not take any space, making the presentation of the material overall cleaner. On the other hand, a math student needs only to click a button to show the proof. The instructor or anyone else reviewing the material may only want to see a few proofs, which is also feasible. The same goes for exercises and their proposed solutions.

Another, more advanced, example of an online textbook is the Stacks Project, available at https://stacks.math.columbia.edu/. One obvious advantage of this approach is the lack of size issues, as the online book takes only some some megabytes instead of some 6000 pages.

Both textbooks have pdf versions as well, which are useful to anyone who wants to have print versions.

In short, the advantages to these online textbooks over traditional ones are:

  • No need to print books and articles (save the trees!);
    • \
  • No need for students to buy books; nor do libraries need to buy several copies of a book (and update them occasionally whenever a new edition is published) to provide them for students who cannot afford those books;
  • Easy access wherever you are; no need to carry books around (save your back!);
  • Some dynamic features which are impossible for hard books;
  • Better referencing and editoring features;
  • The content can be easily modified to fix mistakes, typos, or to add updates and improvements;
  • No issues with the length of the text; The content may be written with as much detail, as many examples, and as pedantically as necessary.
The two possible issues of this approach to textbook are:
  • One needs internet access to read the books; This is remedied by having pdf versions (or really any other type of offline version) available which may be printed out or downloaded.
  • There are server costs with the book webpage; This can only be minimized with decent coding, to make sure the server sends only the necessary data to the readers.
