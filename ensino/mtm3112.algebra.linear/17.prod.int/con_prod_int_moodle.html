Definição.

Um produto interno em um espaço vetorial real \(V\) é uma função

\begin{equation*} \begin{array}{r c c c} \langle\cdot,\cdot\rangle\colon &V\times V &\longrightarrow &\mathbb{R}\\ &(v,w) &\longmapsto &\langle v,w\rangle \end{array} \end{equation*}
satisfazendo às seguintes condições:

  1. Linearidade na primeira coordenada: Para todos \(u,v,b\in V\) e todo \(\lambda\in\mathbb{R}\), vale que

    \begin{equation*} \langle u+\lambda v,b\rangle=\langle u,b\rangle+\lambda\langle v,b\rangle. \end{equation*}
    Isto é, para cada \(b\in V\) fixo, a função
    \begin{equation*} \langle\cdot,b\rangle\colon v\mapsto\langle v,b\rangle \end{equation*}
    é linear.

  2. Simetria: Para todos \(u,v\in V\), vale que

    \begin{equation*} \langle u,v\rangle=\langle v,u\rangle. \end{equation*}

  3. Positividade (estrita): Se \(v\neq 0_V\), então \(\langle v,v\rangle>0\).

Dos axiomas de produto interno, seguem imediatamente as seguintes propriedades:

  1. Linearidade na segunda coordenada: Para cada \(a\in V\) fixo, a função

    \begin{equation*} \langle a,\cdot\rangle\colon v\mapsto\langle a,v\rangle \end{equation*}
    é linear.

  2. Não-degenerecência: \(\langle v,v\rangle=0\) se, e somente se, \(v=0_V\).

  3. Absorvência do zero: \(\langle u,0_V\rangle=\langle 0_V,u\rangle=0\) para todo vetor \(u\in V\).
